PeerSync - Plateforme d'Entraide ENAA
📌 Description

PeerSync est une plateforme de peer-learning développée en PHP orienté objet.
Elle permet aux apprenants de publier des demandes d’aide (tickets) selon différentes technologies, et facilite l’entraide entre étudiants de l’ENAA.

🚀 Fonctionnalités
Création de tickets d’aide
Sélection de technologies :
HTML
CSS
PHP
JAVA
SQL
TAILWIND
JAVASCRIPT
Affichage des tickets dans un dashboard
Filtrage par catégorie de technologie
Architecture orientée objet (OOP)
Validation des données
Système de notation sécurisé
UML et Use Case Diagram
